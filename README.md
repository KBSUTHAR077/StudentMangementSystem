# StudentMangementSystem
In this project the save student data. firstly add new student in database, update student data, delete student data and view students.


#Tools and Technologies
 IDE : Eclipse
 Database: Mysql- Yog
 Core Java 1.7
 Hibernate
 Jsp and Servlet
 Frontend : HTML and CSS
